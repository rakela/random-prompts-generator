## Getting Started

### Installation

#### From Chrome Web Store
1. Visit the [Chrome Web Store](https://chrome.google.com/webstore)
2. Search for "Random Prompts Generator"
3. Click "Add to Chrome"
4. Confirm by clicking "Add extension"
5. The extension icon will appear in your toolbar

#### Developer Installation (For Testing)
1. Download the extension files
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top-right)
4. Click "Load unpacked"
5. Select the `random-prompts-chrome-extension` folder

### First-Time Setup

No setup required! The extension works immediately after installation:
1. Click the extension icon in your toolbar
2. Select a prompt category
3. Click "Generate Prompt"
4. Start creating!

**Tip**: Pin the extension to your toolbar for quick access:
- Click the puzzle piece icon (Extensions)
- Find "Random Prompts Generator"
- Click the pin icon

## Frequently Asked Questions

### General Questions

#### Q: Is Random Prompts Generator really free?
**A:** Yes! The extension is completely free with no:
- Hidden costs
- Subscription fees
- Premium features
- In-app purchases
- Advertisements

#### Q: Do I need an account?
**A:** No account required! Install and start using immediately.

#### Q: Does it work offline?
**A:** Yes! The extension works perfectly without an internet connection. All prompt generation happens locally in your browser.

#### Q: What browsers are supported?
**A:** Currently, the extension supports:
- Google Chrome (recommended)
- Microsoft Edge (Chromium-based)
- Brave Browser
- Other Chromium-based browsers

#### Q: Can I use it on mobile?
**A:** The extension is designed for desktop browsers. Mobile browser extensions have limited support, but some Android browsers like Kiwi may support it.

### Privacy & Data

#### Q: What data do you collect?
**A:** None! We don't collect, track, or transmit any data. Everything stays on your device.

#### Q: Are my prompts private?
**A:** Absolutely! Generated and saved prompts never leave your device.

#### Q: Do you use analytics or tracking?
**A:** No. We don't use Google Analytics, tracking pixels, or any analytics services.

#### Q: Is my data synced across devices?
**A:** No. Saved prompts are stored locally on each device. We don't offer cloud sync.

#### Q: What happens to my data if I uninstall?
**A:** All locally stored data (saved prompts) is deleted when you uninstall the extension.

### Features & Usage

#### Q: How many prompt categories are there?
**A:** Six categories:
1. **Writing** - Story prompts with conflicts and plot twists
2. **AI Art** - Optimized prompts for MidJourney, DALL-E, etc.
3. **Blog** - SEO-friendly blog post ideas
4. **Fantasy** - Worldbuilding prompts with magic systems
5. **Persuasive** - Essay topics and debate ideas
6. **Names** - Fantasy character names

#### Q: Can I customize the prompts?
**A:** You can modify any generated prompt after copying it. Customization of templates isn't available in v1.0 but may be added in future versions.

#### Q: How do I save prompts?
**A:** Click the "Save" button on any generated prompt. Saved prompts appear in the "Saved Prompts" section at the bottom of the popup.

#### Q: Is there a limit to saved prompts?
**A:** No hard limit, but the interface displays the 5 most recent saved prompts. All saved prompts are stored and accessible.

#### Q: Can I export my saved prompts?
**A:** Currently, you can copy prompts individually. Bulk export features may be added in future versions.

#### Q: Can I request new categories?
**A:** Yes! Visit our website to suggest new prompt categories.

### Usage Rights

## Troubleshooting

### Installation Issues

#### Extension Won't Install
**Problem:** "Add to Chrome" button doesn't work or shows an error.

**Solutions:**
1. Update Chrome to the latest version
2. Check your internet connection
3. Clear browser cache and cookies
4. Try a different browser (Edge, Brave)
5. Restart your browser and try again

#### Extension Icon Not Showing
**Problem:** Extension installed but icon doesn't appear in toolbar.

**Solutions:**
1. Click the puzzle piece icon (Extensions menu)
2. Find "Random Prompts Generator"
3. Click the pin icon to pin it to toolbar
4. Or: Right-click the toolbar ‚Üí "Show extension icon"

### Functionality Issues

#### Prompts Not Generating
**Problem:** Clicking "Generate Prompt" does nothing or shows errors.

**Solutions:**
1. **Refresh the extension:**
   - Go to `chrome://extensions/`
   - Find Random Prompts Generator
   - Click the refresh/reload icon
2. **Check browser console:**
   - Right-click the extension popup
   - Click "Inspect"
   - Look for errors in the Console tab
3. **Reinstall the extension:**
   - Remove the extension
   - Restart browser
   - Reinstall from Chrome Web Store

#### Copy Button Not Working
**Problem:** "Copy" button doesn't copy to clipboard.

**Solutions:**
1. **Check permissions:**
   - Go to `chrome://extensions/`
   - Ensure clipboard permission is granted
2. **Browser compatibility:**
   - Update to latest Chrome version
   - Try using keyboard shortcut (Ctrl+C) after selecting text
3. **Manual copy:**
   - Select the prompt text
   - Right-click ‚Üí Copy

#### Saved Prompts Not Appearing
**Problem:** Saved prompts don't show up or disappear.

**Solutions:**
1. **Check storage:**
   - Go to `chrome://extensions/`
   - Click "Details" on Random Prompts Generator
   - Verify storage permission is enabled
2. **Clear and re-save:**
   - Click "Clear All" in Saved Prompts
   - Try saving a new prompt
3. **Check browser storage:**
   - May be a browser storage quota issue
   - Try clearing other extension data

### Display Issues

#### Popup Too Small/Large
**Problem:** Extension popup is too small or large for comfortable use.

**Solutions:**
1. **Browser zoom:**
   - Chrome respects page zoom settings
   - Try Ctrl+0 (zero) to reset zoom
2. **Screen resolution:**
   - Extension is optimized for standard resolutions
   - May need adjustments for very high/low DPI displays

#### Text Overlapping or Cut Off
**Problem:** Text appears overlapping or is cut off.

**Solutions:**
1. **Update extension** to latest version
2. **Reset zoom** to 100% (Ctrl+0)
3. **Report the bug** with a screenshot

### Performance Issues

#### Extension Running Slowly
**Problem:** Prompts take a long time to generate or extension lags.

**Solutions:**
1. **Close unused tabs** to free up browser memory
2. **Disable other extensions** temporarily to check for conflicts
3. **Clear browser cache:**
   - Settings ‚Üí Privacy ‚Üí Clear browsing data
   - Check "Cached images and files"
4. **Update Chrome** to the latest version
5. **Restart browser**

#### High Memory Usage
**Problem:** Extension using too much memory.

**Solutions:**
1. Check Task Manager (Shift+Esc in Chrome)
2. If extension shows high memory:
   - Report the issue with details
   - Reinstall the extension
3. Clear saved prompts if you have many stored

## Privacy & Security

### Data Storage

**What's Stored:**
- Only your saved prompts
- Stored locally using Chrome's storage API
- Never sent to external servers

**Where It's Stored:**
- Your device only
- Chrome's secure local storage
- Isolated from other extensions

### Permissions Explained

**Storage Permission:**
- **Why needed:** To save your favorite prompts
- **What it accesses:** Only data created by this extension
- **What it doesn't access:** Other extension data, browsing history, passwords

### Security Best Practices

1. **Install from Official Source:** Only install from Chrome Web Store
2. **Keep Updated:** Enable automatic updates for security patches
3. **Report Issues:** If you notice suspicious behavior, report immediately
4. **Verify Permissions:** Extension should only request "storage" permission
